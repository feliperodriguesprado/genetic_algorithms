Existe dois projetos que estão construídos na estrutura do Maven:

ga.framework = framework criado pelo professor Artur e adaptado para este trabalho.
production = projeto criado pelos alunos que utiliza o framework para resolver o problema proposto. Após executar esse projeto, o mesmo cria um arquivo result.js dentro da pasta "report\public_html\data" que está na sua raiz. Em seguida, para visualizar o resultado, execute o "arquivo report\public_html\index.html".
